# MarComTeam — Маркетинговые коммуникации

Внутренний портал отдела маркетинговых коммуникаций в стиле Notion: стратегия, OKR, структура, задачи, трекинг и отчётность.

## Разделы

- **Стратегия** — направление и приоритеты отдела
- **OKR** — цели и ключевые результаты с квартальным трекингом
- **Структура** — оргструктура и зоны ответственности
- **Задачи** — трекинг задач, статусы, дедлайны

Обновление информации планируется через AI: сырьё и ссылки на отчёты/разговоры сотрудников отправляются на выделенный e-mail, данные обрабатываются и синхронизируются на сайте.

## Локальный запуск

Откройте `index.html` в браузере или поднимите локальный сервер:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Сайт откроется по адресу `http://localhost:8080`.

## Размещение на GitHub Pages

1. Создайте репозиторий на GitHub (например, `MarComTeam` или `marcomteam`).
2. Загрузите файлы проекта в репозиторий.
3. В настройках репозитория: **Settings → Pages**.
4. В разделе **Source** выберите **Deploy from a branch**.
5. В **Branch** выберите `main` (или `master`), папку — **/ (root)**.
6. Сохраните. Сайт будет доступен по адресу:
   `https://<ваш-username>.github.io/<имя-репозитория>/`

Если репозиторий называется `MarComTeam`, ссылка будет:  
`https://username.github.io/MarComTeam/`
